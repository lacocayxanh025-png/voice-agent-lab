# Upload Checklist

Before publishing this repository:

- [ ] Confirm `.env` files are not present.
- [ ] Confirm there are no real audio files.
- [ ] Confirm there are no real call transcripts.
- [ ] Confirm there are no phone numbers.
- [ ] Confirm browser voice demo does not require provider credentials or real audio samples.
- [ ] Confirm README, roadmap, and issue drafts match the actual repository contents.
- [ ] Confirm the new Git repository history contains no private files, keys, recordings, transcripts, or internal paths from earlier commits.
- [ ] Confirm `npm test` passes.
- [ ] Confirm `npm run privacy:check` passes.
- [ ] Add a GitHub repository description:
  `Privacy-first toolkit for testing voice-agent intent routing, escalation rules, and improvement queues with synthetic conversations.`
- [ ] Add topics:
  `voice-agent`, `intent-routing`, `conversation-design`, `synthetic-data`, `contact-center`, `codex`
- [ ] Publish as a new clean repository, not as a copy of the private working folder.
